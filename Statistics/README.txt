predicate-statistics.xml: Shows in which word-file and axioms each predicate is used. It also contains statistics such as
	1) how many times a predicate is used
	2) for each word-file, in how many axioms this predicate is used 

predicates-for-core-theories.txt: Shows those predicates that were used without definition and are candidates for belonging to core theories. It also shows the line number in the predicate-statistics.xml file where this predicate is used
